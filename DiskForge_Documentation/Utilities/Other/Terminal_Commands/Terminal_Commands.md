# [Terminal_Commands.py](https://faui1-gitlab.cs.fau.de/lena.voigt/diskforge/-/blob/main/DiskForge/Utility/Other/Terminal_Commands.py?ref_type=heads)
The `Terminal_Commands.py` file holds the execution parameters for a multitude of terminal commands used by the framework.

## Commands:
- hexdump
- icat
- remove
- psteal
- find_delete
- touch
- fls
- fsstat
- istat
- mmls
- getconf
- runLogfileCommand
- copy
- getFileEncoding
- checkIfUTF8
- compress
- decompress
- ifind
